## Chitter 
=======

A web-based social app, where users can sign up, log in, post to a feed and view each other's profiles.

### Technologies:
- Ruby
- Sinatra
- PostgreSQL
- HTML
- CSS
- RSpec

### How to run:
Clone this repo, then in the project root directory folder:

Install dependencies
```bash
bundle install
```

Start the server
```bash
rackup
```

Run tests
```bash
rspec
```
